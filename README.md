# Todoplaner
Funktionen:
- Aufgaben hinzufügen
- Aufgaben abhaken
- Aufgaben löschen
- Filter nach Aufgabenart
- Sortieren nach Aufgabentext

Verwendete Technologien:
- Pythonb / Flask (Backend)
- JQuery (für Ajax Anfragen)
- Bootstrap (Frontend) 
- HTML / CSS
